# CampusFlow-ERP
This project is a lightweight, integrated Enterprise Resource Planning (ERP) prototype designed for the higher education sector. The system streamlines core campus operations by digitalizing and interconnecting traditional institutional workflows into a unified platform. By replacing isolated, manual tracking methods, this solution enhances administrative efficiency, minimizes data redundancy, and provides a clear, real-time overview of student and academic lifecycles.

The prototype explicitly implements and integrates four core university modules:
Student Management: Handles student profiles and centralizes basic bio-data.
Course Registration: Maps student profiles to their respective academic modules.  
Attendance Management: Tracks daily lecture attendance and computes eligibility metrics. 
Examination Management: Records, updates, and structures student exam marks per course.  

🛠️ Architecture & Deployment Strategy
Frontend & UI: A responsive, clean dashboard featuring a secure user login interface for system navigation.
Backend & Database: A relational database schema tracking cross-module data dependencies to demonstrate smooth process integration.  
Cloud Infrastructure: Fully deployed and hosted on Microsoft Azure utilizing cloud database and app services to ensure live, public accessibility.
